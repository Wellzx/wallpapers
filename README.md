# Wallpapers

A personal collection of 280+ wallpapers I use across my dotfiles.

## Themes

- Anime
- Minimalist
- Dark
- Nature & Landscape
- Illustration & Art

## Info

- Resolution: 1920x1080 and above
- Format: PNG / JPG

## Usage

Clone the repository and set your wallpaper with your preferred tool:
```
git clone https://github.com/Wellzx/wallpapers.git
```

**Hyprland / Wayland:**
```
swww img wallpapers/wallpaper_name.jpg
```
**X11:**
```
feh --bg-scale wallpapers/wallpaper_ame.jpg
```
